# 神笔PPT

<img width="1175" height="795" alt="image" src="https://github.com/user-attachments/assets/590c3e93-0514-422b-a04c-9a14da0910cb" /><br>

AI 驱动的 PPT 生成工具，输入主题一键生成精美演示文稿。

🔗 **在线体验**：https://shenbippt.netlify.app/

## ✨ 功能特点

- 📝 **智能生成** - 根据用户输入自动生成提纲
- 🎨 **自动配图** - 由提纲生成与之匹配的背景
- 📊 **一键导出** - 一键导出 PPTX 格式文件
- 🎯 **简单易用** - 只需选择或输入主题和页数

## 🛠️ 技术实现

- **框架**：Next.js 16 + React 19
- **样式**：Tailwind CSS 4
- **AI 服务**：SiliconFlow + Qwen 系列模型
- **PPT 生成**：pptxgenjs
- **UI 组件**：Radix UI + Lucide Icons
- **部署**：Netlify / Vercel
